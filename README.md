#Project Titile
AI-Based Task Manager & Smart Scheduler

#Project Summary
An AI-based task management and smart scheduling system that helps users organize their tasks, projects, and goals. The AI automatically creates a manageable schedule, prioritizes work, manages deadlines, and notifies the user about upcoming tasks.
